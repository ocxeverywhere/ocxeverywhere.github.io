Developer Notes

개발 방향

기능을 많이 나열하기보다, 사용자가 앱을 켰을 때 바로 이해하고 끝까지 마칠 수 있는 흐름을 우선합니다. 작은 앱이어도 데이터 처리와 개인정보 안내는 제품의 일부로 다룹니다.

01

반복되는 일을 줄입니다

매일 밤 같은 책을 읽거나, 긴 영상을 계속 바라보는 시간을 앱의 자동화 흐름으로 덜어냅니다.

02

기기 안의 경험을 존중합니다

가능한 데이터는 로컬에 저장하고, 외부 처리가 필요한 기능은 사용자가 이해할 수 있게 설명합니다.

03

작게 출시하고 개선합니다

실제 사용자에게 닿는 앱을 먼저 만들고, 사용 장면이 분명해질수록 기능을 다듬습니다.
